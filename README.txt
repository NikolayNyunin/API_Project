Перед вами прототип простой поисковой системы, реализованной на Python
c использованием API Википедии и API Яндекс.Карт.
При помощи этой поисковой системы можно искать информацию на Яндекс картах или в Википедии.
При поиске по организациям следует вводить запрос в виде:
*Адрес, рядом с которым проводится поиск* + *Тип искомой организации*.
Например: Москва + аптека.
При этом, на карте будут отображены 10 ближайших к заданному адресу организаций,
по данным Яндекс.Карт.
